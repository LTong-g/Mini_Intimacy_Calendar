# MinimalistWeaponEnhancementCalendar

一个专注于“轻量记录、长期复盘”的极简打卡日历应用。

## 这是什么
`MinimalistWeaponEnhancementCalendar` 是一个移动端打卡工具。  
你可以按日期记录三类行为，并在日历与统计图中查看自己的长期变化。

## 你可以做什么
- 按天记录：记录 `观看教程`、`武器强化`、`双人练习` 三类行为。
- 快速浏览：在日视图、月视图、年视图中查看分布与连续性。
- 查看趋势：在统计页查看总览、年度、自定义区间的数据变化。
- 导入导出：通过 JSON 备份或迁移你的打卡记录。

## 适合谁
- 想坚持自律，但不希望使用复杂工具的人。
- 希望回看长期节奏和记录的人。
- 需要本地可控、可导出数据记录的人。

## 产品特点
- 极简交互：核心流程聚焦在“快速记录 + 快速回看”。
- 多尺度视图：日/月/年三个层级覆盖短期与长期观察。
- 本地存储：数据默认保存到本地，支持自主导出。
