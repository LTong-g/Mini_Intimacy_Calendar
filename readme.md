# MinimalistWeaponEnhancementCalendar

一个专注于“记录失控时刻、长期复盘频率”的极简日历应用。

## 这是什么
`MinimalistWeaponEnhancementCalendar` 是一个移动端行为记录工具。
它记录的是三类需要自律、需要降低频率并合理控制的行为，而不是用于坚持完成某项正向习惯。
你可以按日期记录这些行为发生的次数，并在日历与统计图中查看频率变化。

## 你可以做什么
- 按天记录：记录 `观看教程`、`武器强化`、`双人练习` 三类需要控制频率的行为。
- 快速浏览：在日视图、月视图、年视图中查看发生分布与间隔变化。
- 查看趋势：在统计页查看总览、年度、自定义区间的数据变化。
- 导入导出：通过 JSON 备份或迁移你的行为记录。
- 安全锁：可主动开启极简备忘录入口，开启后桌面名称和图标会切换为极简备忘录，启动时先进入极简备忘录，再通过你设置的密码进入记录应用。

## 适合谁
- 想减少某些行为发生频率，并用记录辅助自我控制的人。
- 希望回看长期频率、间隔和变化趋势的人。
- 需要本地可控、可导出数据记录的人。

## 产品特点
- 极简交互：核心流程聚焦在“发生后快速记录 + 长期复盘”。
- 多尺度视图：日/月/年三个层级覆盖短期与长期观察。
- 本地存储：数据默认保存到本地，支持自主导出。
